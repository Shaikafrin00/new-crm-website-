# LeadManager CRM

LeadManager CRM is a modern and responsive lead management web application built to help businesses efficiently manage leads, track customer interactions, and streamline sales workflows through a clean and user-friendly dashboard.

## 🚀 Live Demo

🌐 https://newcrmleadcrm.netlify.app/

---

## 📌 Features

- 📊 Dashboard with lead statistics and overview
- 👥 Add, update, and manage customer leads
- 🔍 Search leads by name, email, or company
- 🏷️ Filter leads by status (New, Contacted, Converted)
- 📅 Track lead follow-ups and updates
- ⚡ Real-time lead status management
- 📱 Fully responsive design for all devices
- 🎨 Clean and modern user interface
